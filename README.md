# 2026 World Cup: Player Efficiency Tracker

A production-grade, end-to-end data engineering pipeline built to process, transform, and serve analytical insights on player performance and disciplinary metrics during the 2026 FIFA World Cup.This project transforms raw, unstructured match event payloads into a clean, star-schema data warehouse model, powering high-level analytics such as normalized per-90 efficiency scores ($G+A / 90$), bench-impact trends, and player discipline risk profiles.
